Personal Finance Analysis

A report exploring transaction behavior, spending patterns, and year-to-year changes in category distributions.

Introduction

This project analyzes personal transaction data to better understand spending behavior across time and categories. The goal is to identify meaningful patterns and communicate them through visual and statistical analysis.

Research Question

How do spending habits differ across years (2023-2026) and categories? Are category spending distributions meaningfully different from one year to another?

Data Overview

The dataset contains transaction-level financial records with variables such as: date, amount, category, whether a record is considered credit/debit, and a description of the transaction (similar to a bank memo).

Key Findings

  • Some weekdays contribute much more to overall spending than others.
  • A few categories make up a large portion of total expenses.
  • Some year pairs have noticeably different category distributions.

Conclusion

This analysis shows that transaction data can reveal meaningful behavioral patterns in spending over time. Future work could compare spending with income, study recurring expenses, or model seasonal effects.
